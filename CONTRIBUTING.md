# Contributing to OpenPS2emulator

Thank you for your interest in the project! This is a work-in-progress, public emulator created for learning purposes.

## How You Can Help
Since I am currently learning to code and building the core logic myself, I am not accepting major code pull requests yet. However, you can still contribute by:

1. **Research:** Sharing documentation on PS2 hardware (EE, GS, VU units).
2. **Bug Reports:** If you try to compile the project and it fails, please open an "Issue."
3. **Suggestions:** If you have ideas for the UI or performance optimizations for mobile hardware (specifically Mali GPUs).

## Rules
* **No AI Code:** Please do not submit code generated by AI. This project is for manual learning.
* **Respect:** Be helpful and patient. We are all learning here!
* **Clean Code:** If you suggest a fix, keep the code readable and commented.

## Getting Started
1. Fork the repository.
2. Create a new branch for your suggestion.
3. Submit an Issue or a Pull Request with a clear description of what you found.

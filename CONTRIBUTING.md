# Contributing to openPS2emulator

Thank you for your interest in improving openPS2emulator! Contributions from the community help make this project better for everyone.

## How to Contribute

### Reporting Bugs
- Use the **Issues** tab to report bugs.
- Provide a detailed description of the hardware you are using (CPU, GPU, RAM).
- Include logs or screenshots if the emulator crashes or encounters a graphical glitch.

### Code Contributions
1. **Fork** the repository and create your branch from `main`.
2. **Implement** your changes. If you are working on specialized drivers (such as Mali-G52 optimizations or Zink/Gladio integration), ensure your code is well-documented.
3. **Test** your changes thoroughly on your local environment.
4. **Submit a Pull Request** with a clear description of what your code solves.

### Testing & Debugging
We highly value feedback from our testers and debuggers. If you are performing stress tests or library shimming, please include the specific build versions and environment details (e.g., Termux environment or specific Android builds) in your report.

---

## Code of Conduct
Please be respectful and professional in all interactions within the project.
